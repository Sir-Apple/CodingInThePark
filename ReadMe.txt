This project is running in PHP Version 7.2.9 with sql Server version: 10.1.35-MariaDB - mariadb.org binary distribution
We suggest to use xampp to run this project
For xampp:
	Place the "RCAL" folder under C:\xampp\htdocs\
	import the "RCSL.sql" in sql folder at http://localhost/phpmyadmin
	and run "http://localhost/rcal/index.php"
